# Ariba OS User Manual

Welcome to **Ariba OS**, your custom AI-powered Linux experience.

## Getting Started

### 1. Installation
To install Ariba OS to your hard drive:
1.  Double-click the **Install Ariba OS** icon on the desktop.
2.  Follow the wizard to select your disk.
    *   **Warning**: This will erase all data on the selected disk.
3.  Once complete, restart your computer.

### 2. First Run (OOBE)
On your first login, the **Ariba Welcome Wizard** will appear.
-   **Language**: Select your preferred language.
-   **Appearance**: Choose between Light and Dark mode.
-   **Ready**: Get direct links to documentation and tools.

## Included Software

Your system comes pre-loaded with essential tools:
-   **Terminal**: `xfce4-terminal` (Access via System Menu or Ctrl+Alt+T).
-   **Editors**: `nano`, `vim`, `mousepad` (GUI).
-   **Web**: `firefox-esr`.
-   **Dev**: `git`, `python3`.
-   **Networking**: Full Wifi/LAN support via Network Manager.

## Custom Tools

### Software Center
Found in the System Menu. Use this to install new applications or ask the AI for recommendations.

### AI Personalizer
Found in Settings. Let the AI analyze your hardware and suggest the best visual settings for you.
